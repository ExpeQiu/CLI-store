  "title": "视频标题",
  "frames": [
    {
      "index": 0,
      "narration": "中文旁白，15-30字",

    }
  ]
}

- 分镜数量 3-5 个，叙事连贯
- narration 使用简体中文
- image_prompt 和 motion_prompt 使用英文
