# 微信客户端直播监控实战清单

## 1. 前置条件

- macOS + 微信 Mac 客户端
- Python ≥ 3.9（**3.14 推荐** `pip install -e ".[capture]"`，使用系统 Vision OCR）
- **系统设置 → 隐私与安全性 → 屏幕录制** → 允许 Terminal / iTerm / Cursor

## 2. 安装

```bash
cd screen-watch-CLI
python3 -m venv .venv && source .venv/bin/activate

# Python 3.14 / 快速开始（macOS Vision OCR，无需下载模型）
pip install -e ".[capture,dev]"

# Python 3.9–3.13 可选 PaddleOCR（中文场景可略好）
pip install -e ".[capture,ocr,dev]"
```

## 3. 一键 E2E 验收

```bash
./scripts/e2e-smoke.sh
```

离线管道 + Vision OCR 测试 + 可选 live 探测（需微信窗口打开）。

## 4. 配置区域（首次）

### 方式 A：截图弹窗框选（推荐）

```bash
# 先列出窗口（如有多个微信窗口）
screen-watch window list --filter 微信

# 打开框选（会弹出独立窗口 + 操作提示）
screen-watch pick --window "微信" --save-config config.yaml

# 多窗口时指定序号，例如直播窗口是 [1]
screen-watch pick --window "微信" --window-index 1
```

**你会看到：**
1. 一个居中弹窗，标题 `screen-watch 框选区域`
2. 顶部绿色提示条：步骤 1/2 框选观看人数
3. 窗口内是微信截图，**按住鼠标拖拽**即可框选

若弹窗被挡住：`Cmd+Tab` 切换到 Python / screen-watch。

### 方式 B：使用默认区域（不框选）

```bash
screen-watch calibrate --window "微信" --defaults --save-config config.yaml
```

## 5. 调试单帧

```bash
mkdir -p logs/debug

# 人数区
screen-watch capture once --region viewer_count --window "微信" -v \
  --save-crop logs/debug/viewer.png

# 弹幕区
screen-watch capture once --region chat --window "微信" -v \
  --save-crop logs/debug/chat.png

# 对已有截图 OCR
screen-watch ocr --input logs/calibrate/wechat-live_xxx.png
```

**OCR 为空时**：打开 `logs/debug/*.png`，对照调整 `regions` 的 x/y/w/h。

## 6. 启动监控

```bash
screen-watch monitor run \
  --window "微信" \
  --interval 1.5 \
  --format jsonl \
  --require-foreground \
  -o logs/$(date +%Y%m%d_%H%M%S).jsonl
```

`--require-foreground`：仅微信在前台时 OCR，避免被其他窗口遮挡。

## 7. 管道入库 zhibo-monitor

```bash
screen-watch monitor run --window "微信" --format jsonl \
  | zhibo-monitor ingest --platform sph-client \
      --event-name "2026 极氪猎装发布会" --car-brand zeekr

# 或
./scripts/pipeline-zhibo.sh
```

入库后：`zhibo-monitor analyze --task-id <id>`

## 8. 常见问题

| 现象 | 排查 |
|------|------|
| 截屏全黑 | 检查屏幕录制权限 |
| OCR 空 | `--save-crop` 看裁剪图；运行 calibrate |
| Paddle 装不上 | Python 3.14 用 `.[capture]` + `--ocr-engine vision` |
| 人数抓不到 | 调整 viewer_count 区域；确认已打开直播页 |
| 弹幕重复 | diff 会过滤；调大 dedup_window |
| CPU 过高 | interval 调至 2–3s；单区域 `--region chat` |
