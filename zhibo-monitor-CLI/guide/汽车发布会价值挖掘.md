# 汽车发布会价值挖掘

## 适用场景

这份清单适用于新能源汽车发布会的信息采集、竞品分析、营销复盘和周报整理。目标不是只记录车型参数，而是从一场发布会里提炼出产品、传播、竞争和市场层面的有效情报。

建议以“一场发布会一张表”的方式固定记录，便于后续做横向对比、时间线跟踪和竞品策略判断。

## 情报表模板

| 字段 | 记录内容 | 填写说明 |
|---|---|---|
| 车型名称 |  | 发布会主推车型全称，可补充版本名、配置名 |
| 品牌定位 |  | 如高端智能、家庭豪华、年轻运动、科技普惠 |
| 目标人群 |  | 如年轻白领、家庭用户、商务人群、女性用户、下沉市场 |
| 核心卖点 Top 3 |  | 只写最核心的 3 个卖点，避免写成参数堆砌 |
| 对标竞品 |  | 明示或暗示对标的车型、品牌或价格带 |
| 传播主叙事 |  | 发布会主线故事，如技术领先、安全第一、家庭幸福 |
| 价格/权益策略 |  | 指导价、预售价、限时权益、金融政策、置换补贴等 |
| 技术亮点 |  | 电池、平台、智驾、芯片、座舱、底盘、补能等核心技术点 |
| 高频关键词 |  | 发布会反复出现的词，如安全、智能、豪华、续航 |
| 发布会最大记忆点 |  | 最容易被传播和用户记住的一个点 |
| 潜在短板 |  | 未讲透、被回避或容易被竞品攻击的点 |

| 对市场格局的影响 |  | 对细分市场、竞品、价格带和用户心智的潜在影响 |

## 字段填写建议

### 1. 车型名称

- 记录车型完整名称。
- 如有必要，补充版本、动力形式和价格带。
- 如果一场发布会发布多款车，可拆分多行记录。

### 2. 品牌定位

- 回答品牌想把自己放在什么位置。
- 重点判断是高端化、走量化、家庭化、科技化，还是年轻化。
- 可从高管表述、发布会主题和视觉风格中提取判断。

### 3. 目标人群

- 回答品牌想打动谁。
- 重点关注年龄层、家庭结构、消费能力、生活方式和使用场景。
- 可从案例用户、视频素材和场景演示中反推。

### 4. 核心卖点 Top 3

- 只保留品牌最想让用户记住的 3 个点。
- 建议优先写“成交点”而不是“配置点”。
- 示例：超充效率、城市智驾、六座家庭空间。

### 5. 对标竞品

- 记录发布会明确提及的竞品。
- 如果没有直接提及，可从价格区间、产品尺寸、功能比较和用户画像推断。
- 对标竞品通常反映品牌想抢谁的用户。

### 6. 传播主叙事

- 回答发布会到底在讲什么故事。
- 常见叙事包括：技术领先、安全可信、家庭幸福、豪华平权、年轻潮流。
- 这个字段与“核心卖点”不同，前者偏传播表达，后者偏产品价值。

### 7. 价格/权益策略

- 不只看指导价，更要看限时权益和成交刺激。
- 重点记录预售价、正式售价、现金优惠、金融政策、置换补贴、赠送配置、首销权益。
- 如果发布会强调“限时”“首批”“下订即享”，通常说明品牌在强化转化。

### 8. 技术亮点

- 聚焦真正构成竞争壁垒的技术能力。
- 可按模块记录：电池、800V、快充、智驾方案、座舱芯片、底盘控制、安全结构。
- 注意区分“自研能力”“合作能力”和“营销包装”。

### 9. 高频关键词

- 统计发布会中反复出现的词。
- 可辅助判断品牌当前最想抢占的认知标签。
- 示例：安全、智能、家庭、豪华、全球、自研、平权。

### 10. 发布会最大记忆点

- 只保留一个最强记忆点。
- 这个点通常最容易被媒体标题化、社媒二次传播和用户复述。
- 例如：超低价格、极限续航、老板名场面、核心技术首发。

### 11. 潜在短板

- 记录品牌没有展开讲、刻意回避或解释过多的内容。
- 也可写容易被竞品反击的点，如定价偏高、补能生态不足、智驾可信度争议、空间不占优。
- 这是后续竞品追踪和舆情监测的重要切口。

### 12. 传播打法判断

- 回答这场发布会在传播上属于什么打法。
- 常见类型：品牌造势、技术占位、订单收割、舆论回应、冲击高端、市场教育。
- 建议结合预热期、发布会当天和发布后扩散节奏一起判断。

### 13. 对市场格局的影响

- 用一句话概括这场发布会可能带来的市场变化。
- 可从价格带冲击、竞品压力、用户心智迁移和细分市场洗牌几个角度判断。
- 示例：将 25 万级家庭 SUV 的竞争重点从空间转向智驾和补能体验。

## 建议补充字段

如果后续要做周报、月报或系统化竞品监测，建议再增加以下字段：

| 字段 | 用途 |
|---|---|
| 发布时间 | 便于按时间线追踪竞品动作 |
| 发布会渠道 | 区分官方直播、车展、门店活动、跨界联名等 |
| 声量表现 | 记录热视频、热搜、媒体报道量、互动量等 |
| 转化动作 | 记录试驾、下订、盲订、锁单、权益截止时间 |

## 使用建议

- 一场发布会填一行，形成统一数据库。
- 每个字段尽量写“判断”而不是只写“事实”。
- 可在发布会结束后 24 小时内完成初版，随后再补充舆情和销量反馈。
- 重点关注“核心卖点”“传播主叙事”“潜在短板”“市场影响”四个字段，它们最有分析价值。
