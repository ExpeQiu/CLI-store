```yaml
comfyui:
  endpoint: http://127.0.0.1:8188
```

### 服务器管理命令

```bash
# 初始化（首次，将 ComfyUI 移至数据盘）
bash /root/tzwm-scripts/ComfyUI/autodl/scripts/init_comfyui.sh

# 启动 ComfyUI
source /etc/network_turbo
export PATH=/root/miniconda3/bin:$PATH
cd /root/autodl-tmp/ComfyUI
nohup python main.py --port 6006 --listen 0.0.0.0 > /root/autodl-tmp/logs/comfyui_start.log 2>&1 &

# 查看日志
tail -f /root/autodl-tmp/logs/comfyui_start.log

# 下载模型（HuggingFace，支持断点续传）
bash /root/autodl-tmp/download_models.sh          # Z-Image-Turbo
bash /root/autodl-tmp/download_wan_flux.sh        # Wan 2.1 + FLUX
tail -f /root/autodl-tmp/logs/download_wan_flux.log
```

### 模型目录

| 路径 | 用途 |
|------|------|
| `/root/autodl-tmp/models/diffusion_models/` | 扩散模型（Wan / FLUX / Z-Image） |
| `/root/autodl-tmp/models/vae/` | VAE |
| `/root/autodl-tmp/models/text_encoders/` | 文本编码器（Wan umt5 等） |
| `/root/autodl-tmp/models/clip/` | CLIP（FLUX clip_l / t5xxl） |
| `/root/models/` | 系统盘模型（upscaler 等） |

> 数据盘仅 50GB，不宜同时保留 Z-Image + Wan + FLUX 全量模型。

> 注意：镜像内置 `cg down` 下载器在当前实例不可用，模型通过 HuggingFace 下载脚本获取。
